lineNumbers

It's simple - count the line numbers on your Google Document - fast. Download Now

About

After being inspired by our Spanish teacher, we became aware that Google had no add on that would manually edit a document to add line numbers. So we decided to fix that.

Scopes

../auth/documents is used to edit the document and download the document content for line number detection.

../auth/script.container.ui is used to add the add-on to the ui bar.

../auth/script.external_request is used to call an API that gets the number of lines in the document
